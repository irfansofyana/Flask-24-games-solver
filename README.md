# Flask-24-games-solver

## How to Run
- Make sure you have python and pip in your system.
- Make sure you have installed flask. If you haven't install yet, you can install it with command:
  for python3:

  ```
  pip3 install flask
  ```
  
  for python2:
  
  ```
  pip install flask
  ```
- Run app with command:
  for python3:
  
  ```
  python3 main.py
  ```

  for python2:

  ```
  python main.py
  ```
